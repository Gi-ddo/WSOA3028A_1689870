What is a URL?

A uniform resource loader (URL) is a unique identifier to find resources on the internet. It is often called the web address of a website and it consists of different properties mainly called the protocol and the domain name. These main function is telling the browsers where and how to retrieve information or a resource of the WWW.


URL structure


As mentioned above the URL contains the protocol and the domain name. The protocol is what is needed to access information or a resource URL protocols include HTTP and HTTPS for mainly web resources and mail addresses. For file transfer, the FTP protocol is responsible for that functionality. The second part of the URL is responsible for identifying domain names and IP addresses. This tells us where the resource is loaded. The URL also specifies a few key bits of information including:

  • The path to a specfic page or file domain
  • The network port to where the connection is made
  • Search parameters querys, this is most commomly used for search results

Parts of URL

Here is a summary of the most important parts of the URL:

  • The protocol: Used for access to resources on the internet.
  • Domain name: Unique reference to a website.
  • Port name: Port for web browers.
  • Path: Location on web server.
  • Query: Search requests.
