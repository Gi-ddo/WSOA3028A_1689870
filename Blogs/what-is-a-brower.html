What is a brower? How do they work?

A browser is software application used to retrieve, display and store content from the world wide web. These types of contents include web pages, images, videos and music etc. All the digital medium we have access to is mostly received through a browser, these browsers are accessed through our phones and computers. The most widely known browsers are: Chrome, Firefox, Internet explorer, Safari and Opera.

Browers work in hierarchical manner to process all the information we request and feed it back to us on our computers and phones. The structure is as follows:

  • User interface: The user interface is what users interact with when using their browsers. It has the search bar, the bookmark options, your history and all the different types of information that is available to users in the browsers we use today.

  • Browers engine: The browser engine serves as a medium between the user interface and the and what is called the rendering engine, all input by the users is processed and shown back to us on the screen.

  • Rendering engine: The rendering engine is responsible for processing all the html and CSS to form what we see on the screen when we get back results from our searches.

  • Networking: Networking is responsible for receiving all the URLs using protocols of Http and networking is responsible for dealing with all communications through the internet and handles how quickly we get our information depending on internet traffic.

  • UI backend: The UI backend is used for drawing widgets for our browsers and the windows that we use to type our search requests.

  • JavaScript interpreter: This deals with link between the JavaScript written to give functionality to the website.

  • Data storage: As the name suggest this is where all data storage takes place, with things like cookies, file systems and local storage is being handled.
